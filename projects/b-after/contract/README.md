# B-After 月額保守契約 テンプレ一式

2026/04/21 作成

## ファイル構成

| ファイル | 用途 | 渡す相手 |
|---|---|---|
| `01_保守業務仕様書.md` | 作業範囲・条件の取り決め（一番重要） | クライアント |
| `02_見積書.md` | 金額提示 | クライアント |
| `03_発注書.md` | クライアントから返送してもらう用 | クライアント記入 |
| `04_請書.md` | 発注書受領後にこちらから返送 | こちら発行 |
| `05_提案メール文面.md` | 最初の提案メール本文 | メール送信時コピペ |

## 使い方

### ステップ1：各ファイルの〇〇部分を埋める
- 日付
- 住所（〒、住所）
- 振込先銀行口座
- 契約期間の具体日

### ステップ2：Word/PDFに変換
現在は Markdown。以下のどちらかで整形：
- **Notion / Google Docs に貼り付け** → PDF出力（カンタン）
- **Word を開いて貼り付け** → スタイル調整 → PDF出力
- **`pandoc` で一括変換**（コマンドラインで）

### ステップ3：送付
メール文面（05）をベースに、1〜3をPDFで添付して送付。

### ステップ4：契約成立まで
1. クライアントが発注書（03）に記入・捺印して返送
2. こちらから請書（04）を発行して送付
3. 翌月1日から保守開始

## 重要な契約条件

- 最低契約期間：**6ヶ月**
- 自動更新：1ヶ月ごと
- 解約通知：1ヶ月前
- 作業枠：月4時間まで、超過分 4,000円／時
- 損害賠償上限：月額保守料の1ヶ月分
- 免責：サーバー提供元・第三者製プラグインの不具合
